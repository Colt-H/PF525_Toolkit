# PF525 Toolkit

A collection of various Python3 based tools for working with Allen Bradley Powerflex 525 Variable Frequency Drives.

## Roadmap

1. Powerflex 4/40/40P Parameter Cross Reference
2. Parameter Help: Notes, related parameters, wiring, and example applications
3. Drive Connected Tools: Get parameter, set parameter, get statuses, and more
4. L5X Generation: Name a drive, and generate a basic set of stop/start/fault handling logic

## Requirements

- Pycomm3 V1.0.0
- SQLite3
